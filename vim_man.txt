[escape] -> [:][q] - выйти
[escape] -> [:][q][!] - выйти без сохранения

[/] - поиск

    [n] - искать дольше
    [shift][n] - искать выше

vim -R <file name> - открыть только для чтения

Режимы работы:

    Коммандный
        
        [escape]

    Коммандной строки - [:]

        [q] - выйти
        write - записать
        set nonumber - убрать нумерацию строк
        set number - включить нумерацию строк

    Ввода
        
        [i] - вставка до курсора
        [a] - вставка после курсора

    Вмзуальный

        [v]
